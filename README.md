## 3. ü§∏ Quick Start (Local Development)<a id="quick-start"></a>

###  Prerequisites

Make sure you have the following installed:

| Tool | Tested Version | Notes |
|------|----------------|-------|
| **Node.js** | ‚â• 18.x | <https://nodejs.org/> |
| **Npm**  | any | <https://docs.npmjs.com/> |
| **Git** | any | <https://git-scm.com/> |

> Using **pnpm** is recommended for its speed and disk-space savings, but npm or yarn will work too.

###  Clone the Repo

```bash
git clone https://github.com/rr3s1/portfolio.git
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run the Dev Server

```bash
npm run dev
```

Open http://localhost:3000 in your browser ‚Äì the app will hot-reload on changes.
